# React + TypeScript + Vite + Tailwind

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

1. git clone "http link for the repo"
2. cd react-boilerplate
3. npm install
4. npm run dev

## API URL Configuration

The API base URLs for development and production are hardcoded in `src/api.ts`:

- **Development backend:** https://meshspire-core-vjqd.onrender.com/api/v0
- **Production backend:** https://meshspire-core-prod.onrender.com/api/v0

No `.env` files are required or used for API URLs in deployment. Do not add `.env`, `.env.dev`, or `.env.prod` to the deployed frontend.
